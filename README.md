
# Flutter Product Listing

 Flutter Product Listing & Filter using Getx.  


## Run Locally

Clone the project

```bash
  git clone https://github.com/Hisham-reactz/flutterMT_getx.git
```

Go to the project directory

```bash
  cd flutterMT_getx
```

Install dependencies

```bash
  flutter create .
```

Run the app

```bash
  flutter run
```


## Features

- Product Listing
- Product Filter
